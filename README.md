<h2>🚀 Overview</h2>

<p>
<b>EcoShop GUI X</b> is a modern, high-performance Minecraft shop plugin designed for 
<b>premium servers</b>. Built for <b>Paper 1.20+</b>, it delivers a fully interactive GUI-based shop system 
powered by <b>Vault + EssentialsX</b>.
</p>

---

<h2>✨ Features</h2>

<h3>🛒 Advanced GUI Shop</h3>
<ul>
<li>Clean and modern interface</li>
<li>Category-based navigation</li>
<li>Buy & Sell system</li>
<li>Pagination support</li>
</ul>

<h3>💰 Economy Integration</h3>
<ul>
<li>Works with <b>Vault</b> + <b>EssentialsX</b></li>
<li>Plug & play setup</li>

</ul>

<h3>📈 Dynamic Pricing</h3>
